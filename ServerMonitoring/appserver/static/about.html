"Server Inter-dependency For Splunk" App integrates the open source monitoring solution Nagios with Splunk and uses D3 for visualization of dependency between servers

Features

  • RealTime server status from Nagios
  • Visual Representation of Infrastructure Hierarchy
  • Determine System Resiliency
  • Ensure System Availability
  • D3 Tree Interactive Integration with Splunk
